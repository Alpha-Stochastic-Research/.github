# Support

Thank you for your interest in **Alpha Stochastic Research (ASR)**.

This page explains how to get help, ask questions, report issues, or contact ASR regarding research, open-source projects, documentation, and community participation.

---

## General Questions

For general questions about ASR projects, please use GitHub Issues or Discussions when available.

Use the **Question** issue template for:

- Questions about repository usage
- Installation help
- Python implementation details
- Mathematical derivations
- Numerical simulations
- Research concepts
- Documentation clarification
- Reproducibility questions

Before opening a new question, please search existing issues and discussions.

---

## Bug Reports

For bugs, errors, unexpected outputs, broken workflows, or incorrect numerical behavior, please open a GitHub issue using the **Bug Report** template.

When reporting a bug, include:

- The affected repository
- A clear description of the issue
- Steps to reproduce
- Expected behavior
- Actual behavior
- Error messages or logs
- Python version and operating system
- Relevant parameters, equations, or numerical settings

For scientific or numerical bugs, please include enough context to reproduce the result.

---

## Feature Requests

For proposed improvements, new implementations, documentation updates, notebooks, figures, or workflow enhancements, please use the **Feature Request** template.

Good feature requests should explain:

- The problem or limitation
- The proposed solution
- The scientific or technical value
- Possible alternatives
- References, if relevant

---

## Research Proposals

For new research directions, paper reproductions, mathematical models, numerical experiments, or open-science projects, please use the **Research Proposal** template.

Research proposals should include:

- A clear title
- Motivation
- Research area
- Methodology
- Expected outputs
- Reproducibility plan
- References
- Known limitations or assumptions

---

## Security Issues

Please do not report security vulnerabilities through public GitHub issues.

For security concerns, exposed secrets, dependency risks, or unsafe configurations, contact ASR privately:

**research@asr-lab.online**

Please use the subject line:

```text
[Security] Vulnerability Report - Alpha Stochastic Research
```

See `SECURITY.md` for more details.

## Response Expectations

ASR is an independent research initiative. Response times may vary depending on the nature of the request, the complexity of the topic, and current research priorities.

For the best chance of receiving a useful response:

- Be precise
- Provide context
- Include reproducible examples
- Add references when relevant
- Use the correct issue template
- Keep discussions professional and focused
