# Alpha Stochastic Research

**Alpha Stochastic Research (ASR)** is an independent quantitative research laboratory working on financial mathematics, stochastic modelling, risk management, portfolio research, financial machine learning and reproducible scientific computing.

- Website: https://www.asr-lab.online
- Research: https://www.asr-lab.online/research/
- Events: https://www.asr-lab.online/events/
- Open Lab: https://www.asr-lab.online/lab/
- Join Us: https://www.asr-lab.online/join/
- Contact: research@asr-lab.online

## Principles

1. Rigorous research.
2. Reproducible methods.
3. Transparent assumptions and limitations.
4. Responsible collaboration.
5. Open science whenever legally and ethically possible.
